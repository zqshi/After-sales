## 2.3 知识应用功能

> **PRD格式**: Feature PRD (8章标准格式)
> **优先级**: P1 (重要功能)
> **所属版本**: v1.0
> **最后更新**: 2026-01-14

---

### 第1章：功能概述

#### 1.1 功能定位

**知识应用**用于在处理对话时快速检索并复用知识内容，通过关键词检索与场景快捷入口，将知识摘要与正文展示给客服，支持一键复制。

**核心价值**:
- ✅ 场景化检索，快速定位知识
- ✅ 展示摘要与正文，便于内部参考
- ✅ 一键复制，便于粘贴到回复草稿

#### 1.2 目标用户

| 用户角色 | 使用场景 | 核心需求 |
|---------|---------|---------|
| 一线客服 | 处理客户问题 | 快速找到处理步骤并复制内容 |
| 主管/培训 | 复盘与培训 | 查找标准处理话术与步骤 |

#### 1.3 功能范围

**本期范围**:
- Dock → 知识 → 知识应用入口
- 关键词检索（模糊匹配）
- 场景快捷入口（登录/账号锁定/系统报错等）
- 检索结果列表 + 匹配度展示
- 详情侧栏（摘要/正文/标签）
- 一键复制内容

**不在本期范围**:
- 直接引用到对话输入框
- 语义向量检索与实时索引
- 使用统计/评分
- 知识推荐自动推送到对话区

### 第2章：用户场景

#### 场景1：检索登录问题

**用户旅程**:
```
1. Dock进入“知识应用”
2. 输入关键词“登录失败”或点击场景快捷入口
3. 系统返回匹配结果与匹配度
4. 点击“查看详情”查看摘要与正文
5. 点击“复制内容”复制处理步骤
```

---

### 第3章：功能详细设计

#### 3.1 检索区

- 输入框：支持关键词描述
- 快捷入口：8个场景按钮
- 清空按钮：清除关键词

#### 3.2 结果区

- 卡片列表：标题、分类、摘要、标签、匹配度
- 详情区：正文、标签、更新时间
- 复制按钮：复制当前详情内容

---

### 第4章：交互设计

- 搜索后展示结果列表与详情侧栏
- 未命中显示空状态提示
- 复制操作成功后反馈提示

### 第5章：数据模型

```typescript
interface KnowledgeSearchItem {
  id: string;
  title: string;
  category: string;
  summary: string;
  content: string;
  tags: string[];
  score: number; // 0-1 匹配度
  updatedAt: string;
}
```

---

### 第6章：业务规则

- 匹配逻辑基于关键词与语义映射
- 检索结果按匹配度降序展示

- 结果列表滚动平滑
