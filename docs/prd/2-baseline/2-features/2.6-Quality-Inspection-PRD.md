## 2.6 质检功能

> **PRD格式**: Feature PRD (8章标准格式)
> **优先级**: P0 (核心功能)
> **所属版本**: v1.0
> **最后更新**: 2026-01-14

### 实现状态（当前基础设施对齐）

- InspectorAgent 当前仅支持基于对话文本的质检评分输出，缺少后端MCP质检工具与报告链路。
- 质检触发、报表与申诉流程需后端工具与数据模型补齐后落地。

---

### 第1章：功能概述

#### 1.1 功能定位

**质检功能**用于质量专员在任务工作区对群聊会话进行质检复盘，展示质检摘要、维度评分与优化建议，并可在右侧抽屉打开质检详情。

**核心价值**:
- ✅ 质检概览与客户列表
- ✅ 质检详情（维度评分/对话节选/洞察）
- ✅ 与客户详情联动

#### 1.2 目标用户

| 用户角色 | 使用场景 | 核心需求 |
|---------|---------|---------|
| 质检专员 | 会话复盘 | 快速查看质检摘要与问题点 |
| 主管/经理 | 质量巡检 | 了解高紧急对话与建议 |

#### 1.3 功能范围

**本期范围**:
- 任务工作区“质量检测分析”概览
- 客户质量状态列表
- “对话质检”按钮打开右侧质检面板
- 质检维度评分、对话节选与洞察展示
- 视图切换（综合/质检分析/对话复盘）

**不在本期范围**:
- 评分算法配置与编辑
- 质检报告导出
- 质检任务派发与审批流
- 抽检规则配置与抽检策略管理

### 第2章：用户场景

#### 场景1：质检专员查看质检概览

**用户旅程**:
```
1. 切换到任务工作区
2. 查看质检概览指标与客户列表
3. 点击“对话质检”打开右侧面板
```

#### 场景2：查看质检详情

**用户旅程**:
```
1. 打开右侧质检面板
2. 查看摘要与标签
3. 查看三维度评分与对话节选
4. 阅读洞察与优化建议
```

---

### 第3章：功能详细设计

#### 3.1 质检概览

- 指标卡片：平均质检得分/高紧急会话/自动派发指令
- 客户列表：会话摘要 + 质检分数 + 操作按钮

#### 3.2 质检详情面板（右侧抽屉）

- 头部摘要：会话标题/紧急度/渠道/更新时间
- 维度评分：响应时效/情感态度/回复质量
- 对话节选：关键片段
- 洞察与优化建议

#### 3.3 质检维度

- 响应时效：首响时间、回复节奏
- 情感态度：礼貌用语、共情表达
- 回复质量：解决步骤完整性与可执行性

---

### 第4章：交互设计

- “对话质检”打开右侧抽屉并切换至质检视图
- 视图切换按钮仅影响展示分区
- “客户详情”返回经典分析面板

### 第5章：数据模型

interface QualityProfile {

---

### 第6章：业务规则

#### 6.1 质检触发规则

- 由大模型判断会话内一次任务被完整解决后触发质检任务

#### 6.2 质检范围

- 范围：群聊会话及其关键节点
- 覆盖：客户消息与客服回复的完整上下文

#### 6.3 评分与阈值

- 维度评分统一为百分制
- 低分对话进入复盘清单并生成优化建议
