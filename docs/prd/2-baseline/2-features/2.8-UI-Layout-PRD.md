## 2.8 UI布局功能

> **PRD格式**: Feature PRD (8章标准格式)
> **优先级**: P0 (核心功能)
> **所属版本**: v1.0
> **最后更新**: 2026-01-14

---

### 第1章：功能概述

#### 1.1 功能定位

**UI布局**采用四栏结构：Dock导航栏 + 左侧边栏 + 主内容区 + 右侧抽屉面板，保障对话处理与辅助信息同屏展示。

**核心价值**:
- ✅ Dock导航切换核心模块
- ✅ 左侧列表与主内容联动
- ✅ 右侧抽屉承载分析/质检/AI辅助
- ✅ 支持左右侧宽度拖拽调整

#### 1.2 目标用户

| 用户角色 | 使用场景 | 核心需求 |
|---------|---------|---------|
| 客服人员 | 日常对话处理 | 快速切换对话与查看客户信息 |
| 质检专员 | 质检复盘 | 切换质检面板查看详情 |
| 管理层 | 数据查看 | 查看报表中心 |

#### 1.3 功能范围

**本期范围**:
- Dock导航栏（消息/知识/工具/报表）
- 左侧边栏（对话/任务）
- 主内容区（对话/知识/工具/报表/任务）
- 右侧抽屉（经典分析/AI辅助/质检）
- 左右侧宽度拖拽

**不在本期范围**:
- 布局自定义与持久化
- 多标签页与分屏
- 深色主题

---

### 第2章：用户场景

#### 场景1：客服处理对话

**用户旅程**:
```
1. 打开工作台默认进入消息-对话
2. 左侧选择对话，主内容区加载聊天
3. 点击“分析面板”打开右侧抽屉
4. 需要AI建议时切换到AI辅助面板
```

#### 场景2：质检专员查看质检

**用户旅程**:
```
1. 切换到任务工作区
2. 点击“对话质检”
3. 右侧抽屉切换为质检面板
```

---

### 第3章：功能详细设计

#### 3.1 布局结构

```
Header (56px)
Dock (64px) | 左侧边栏 (可拖拽) | 主内容区 | 右侧抽屉 (可拖拽/可收起)
```

#### 3.2 Dock导航

- 消息：对话/任务
- 知识：知识管理/知识应用
- 工具：工具中心
- 报表：报表中心

#### 3.3 右侧抽屉模式

- 经典分析面板（客户信息/历史记录）
- AI辅助面板（回复建议/解决方案/需求检测）
- 质检面板（维度评分/对话节选）

---

### 第4章：交互设计

- Dock切换后主内容区切换
- 右侧抽屉可打开/关闭
- 左右侧可拖拽调整宽度

---

### 第5章：数据模型

```typescript
interface LayoutState {
  leftSidebarWidth?: number;
  drawerWidth?: number;
  drawerMode?: 'analysis' | 'ai' | 'quality';
}
```

---

### 第6章：业务规则

- 右侧抽屉默认关闭
- 知识/工具/报表模式下右侧抽屉关闭

---

### 第7章：非功能需求

- 布局切换 < 300ms
- 拖拽调整响应流畅

- [ ] 右侧抽屉可开关与切换
