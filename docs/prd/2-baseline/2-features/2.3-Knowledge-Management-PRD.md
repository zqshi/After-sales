## 2.3 知识管理功能

> **PRD格式**: Feature PRD (8章标准格式)
> **优先级**: P0 (核心功能)
> **所属版本**: v1.0
> **最后更新**: 2026-01-14

---

### 第1章：功能概述

#### 1.1 功能定位

**知识管理**用于维护知识文档与FAQ，提供分类树、搜索、上传、查看、删除等基础管理能力。

**核心价值**:
- ✅ 文档/FAQ双库管理
- ✅ 分类树与关键词搜索
- ✅ 详情查看与删除确认
- ✅ FAQ相似问题维护

#### 1.2 目标用户

| 用户角色 | 使用场景 | 核心需求 |
|---------|---------|---------|
| 知识管理员 | 维护知识库 | 上传/整理文档与FAQ |
| 一线客服 | 查询并复用 | 快速定位文档或FAQ |

#### 1.3 功能范围

**本期范围**:
- Dock → 知识 → 知识管理入口
- 分类目录树（最多3层）
- 文档库：搜索/列表/详情/删除
- FAQ库：搜索/新增/编辑/删除
- 文档上传弹窗（选择分类、文件列表）
- FAQ相似问题维护（手动添加/生成）

**不在本期范围**:
- 知识版本管理、审批流程
- 评分/引用统计
- 批量导入与同步外部知识源

### 第2章：用户场景

#### 场景1：上传知识文档

**用户旅程**:
```
1. 进入“知识管理”
2. 点击“上传文档”
3. 选择分类与文件
4. 提交后文档进入文档库列表
```

#### 场景2：维护FAQ

**用户旅程**:
```
1. 切换到FAQ标签
2. 点击“新增FAQ”
3. 填写问题/答案/状态
4. 维护相似问题列表并保存
```

---

### 第3章：功能详细设计

#### 3.1 分类树

- 左侧目录树
- 支持关键词筛选
- 面包屑展示当前分类

#### 3.2 文档库

- 字段：标题、分类、状态、更新时间、操作
- 状态：active / archived / deprecated
- 详情弹窗展示来源、负责人、摘要、正文

#### 3.3 FAQ库

- 字段：问题、状态、更新时间、操作
- 支持新增/编辑/删除
- 支持相似问题维护与检测

---

### 第4章：交互设计

- 文档/FAQ Tab切换
- 上传与删除采用弹窗确认
- 空列表显示引导提示

---

### 第5章：数据模型

```typescript
interface KnowledgeDoc {
  id: string;
  title: string;
  status: 'active' | 'archived' | 'deprecated';
  summary?: string;
  content: string;
  category: string;
  tags: string[];
  owner?: string;
  source?: string;
  createdAt?: string;
  updatedAt?: string;
}

interface KnowledgeFaq {
  id: string;
  question: string;
  answer: string;
  status: 'active' | 'archived' | 'deprecated';
  similarQuestions?: string[];
  createdAt?: string;
  updatedAt?: string;
}
```

### 第6章：业务规则

- 文档状态仅允许 active/archived/deprecated
- FAQ相似问题支持手动添加与自动生成
