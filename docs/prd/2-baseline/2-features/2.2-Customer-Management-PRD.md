## 2.2 客户信息面板功能

> **PRD格式**: Feature PRD (8章标准格式)
> **优先级**: P0 (核心功能)
> **所属版本**: v1.0
> **最后更新**: 2026-01-14

---

### 第1章：功能概述

#### 1.1 功能定位

**客户信息面板**为对话处理提供客户画像视图，作为右侧经典分析面板的一部分，展示客户基础信息、合同指标与历史对话。

**核心价值**:
- ✅ 对话上下文中快速查看客户画像
- ✅ 展示客户等级、合同周期与关键指标
- ✅ 历史对话时间线可定位到当前会话

#### 1.2 目标用户

| 用户角色 | 使用场景 | 核心需求 |
|---------|---------|---------|
| 一线客服 | 对话处理中快速识别客户信息 | 了解等级、合同与购买产品 |
| 主管/经理 | 复盘对话 | 查看历史记录与客户摘要 |

#### 1.3 功能范围

**本期范围**:
- 右侧抽屉“客户信息”Tab
- 客户基础信息（姓名/职位/联系方式）
- 客户等级与有效期（VIP/KA0/KA1）
- 合同周期与关键指标（合同金额/满意度/合作时长）
- 已购产品列表
- 群公告与群备注
- 当前客情摘要
- 历史对话时间线（支持定位到对话）

**不在本期范围**:
- 客户列表/客户管理入口
- 客户信息编辑、标签管理
- 合同管理与续约提醒
- 客户分群/评分/流失预警

### 第2章：用户场景

#### 场景1：对话中查看客户画像

**用户旅程**:
```
1. 在对话页点击“分析面板”
2. 进入右侧“客户信息”Tab
3. 查看客户等级、合同指标、联系方式
4. 阅读群公告与群备注
5. 根据客情摘要调整回复节奏
```

#### 场景2：复盘历史对话

**用户旅程**:
```
1. 切换到“历史对话记录”Tab
2. 选择时间范围
3. 查看时间线中的关键沟通
4. 点击“定位对话”回到当前对话内容
```

---

### 第3章：功能详细设计

#### 3.1 客户信息卡片

展示内容:
- 客户名称、职位、联系方式
- 客户等级与有效期
- 合同周期与关键指标
- 已购产品列表

#### 3.2 群信息与客情摘要

- 群公告：面向客户的最新公告 **（是否需与三方IM渠道打通需确认）**
- 群备注：内部协作备注 **（是否需与三方IM渠道打通需确认）**
- 客情摘要：当前风险/情绪/建议

#### 3.3 历史对话记录

- 时间线列表
- 支持“定位对话”跳转
- 时间范围筛选（最近30天/90天/全部）

---

### 第4章：交互设计

- 入口：右侧抽屉 → 经典分析面板
- Tab切换：客户信息 / 历史对话记录
- “定位对话”将主对话区滚动至对应消息
