# 测试用例库

> **文档版本**: v1.0
> **更新日期**: 2026-01-27

---

## 1. 功能模块用例清单

### 1.1 会话与消息
- 创建会话、查询会话、关闭会话
- 发送消息、消息回放、异常输入处理
- WebSocket 订阅（如适用）

### 1.2 需求管理
- 创建需求、状态流转、权限校验
- 需求列表筛选、分页、导出

### 1.3 任务管理
- 创建任务、分配、完成、关闭
- 任务并发更新（乐观锁）

### 1.4 质检与审计
- 质检报告生成与查询
- 审计事件记录与查询

### 1.5 知识库
- 知识条目 CRUD
- 搜索与推荐

---

## 2. 通用校验用例

- 必填字段缺失
- 枚举值非法
- 权限不足（401/403）
- 并发更新冲突（409）
